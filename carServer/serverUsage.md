# carserver usage

## get all car

```
http://localhost:3000/car
```

### get all types

```
http://localhost:3000/car

```

### get model by license

```
http://localhost:3000/search/bylicence?license='a'

```

### search by model

```
http://localhost:3000/search/byYear?year=11818
```
